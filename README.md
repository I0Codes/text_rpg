# 🎮 Текстова RPG

[![Python](https://img.shields.io/badge/Python-3.x-blue.svg)](https://www.python.org/)

> Текстова рольова гра у стилі "A Dark Room" — пориньте у світ пригод та виживання! 🌲

## 📖 Про гру

**Текстова RPG** — це захоплююча текстова пригода, де ви прокидаєтеся в таємничому лісі і повинні виживати, досліджувати світ та розвивати свого персонажа.

### ✨ Особливості

- 🎭 **Три унікальні класи персонажів:**
  - ⚔️ **Воїн** — майстер ближнього бою
  - 🔮 **Маг** — володар магічних сил
  - 🏹 **Розвідник** — швидкий та спритний мисливець

- 🗺️ **Геймплей:**
  - Досліджуйте таємничий світ
  - Виживайте в суворих умовах
  - Битеся з ворогами
  - Прогресивна складність

## 🚀 Як запустити

### Вимоги

- Python 3.x

### Запуск гри

```bash
python main.py
```

## 📁 Структура проекту

```
text_rpg/
```
